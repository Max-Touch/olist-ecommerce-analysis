# Анализ продуктовых метрик — интернет-магазин косметики

## О проекте
Продуктовый анализ данных реального интернет-магазина косметики на основе событий пользователей за 5 месяцев (октябрь 2019 — февраль 2020). Цель — рассчитать ключевые продуктовые метрики и выявить проблемы с удержанием пользователей.

## Данные
Источник: [E-Commerce Events History in Cosmetics Shop — Kaggle (Rees46)](https://www.kaggle.com/datasets/mkechinov/ecommerce-events-history-in-cosmetics-shop)

- ~20 млн событий пользователей за 5 месяцев
- Типы событий: view, cart, remove_from_cart, purchase
- Колонки: event_time, event_type, product_id, category_code, brand, price, user_id, user_session

## Стек
- **PostgreSQL** — хранение данных (~20 млн строк)
- **SQL** — агрегации, CTE, оконные функции, VIEW
- **Python / pandas** — обработка результатов и расчет метрик
- **seaborn / matplotlib** — визуализация
- **statsmodels** — статистическое тестирование (Z-тест)

## Что анализировали
1. Воронка продаж (view → cart → purchase)
2. DAU — дневная активность пользователей
3. MAU — месячная активность по всем 5 месяцам
4. Retention — когортный анализ
5. LTV — пожизненная ценность пользователя
6. Churn Rate — отток по когортам
7. Топ бренды и категории по выручке
8. A/B тест — конверсия в будни vs выходные (Z-тест для пропорций)

## Ключевые результаты

| Метрика | Значение |
|---|---|
| Конверсия view→purchase | 8.86% |
| MAU | 370-410к пользователей |
| Retention после месяца 1 | 9-14% |
| Churn Rate месяц 1 | ~88% |
| Медианный LTV | 33 у.е. |
| Топ бренд | runail |
| Топ категория | appliances |
| Конверсия в будни | 7.1% |
| Конверсия в выходные | 6.0% |

## Главные выводы

1. **Критически низкий Retention** — 88% пользователей не возвращаются после первого месяца. Это главная проблема бизнеса.
2. **Узкое место воронки** — переход view→cart (26.98%). Большинство пользователей уходят не добавив товар в корзину.
3. **Рост аудитории отсутствует** — MAU стабилен на уровне 370-410к, без тренда роста.
4. **Будни эффективнее выходных** — конверсия в будни (7.1%) статистически значимо выше чем в выходные (6.0%). P-value ≈ 0.
5. **LTV низкий** — медианный пользователь тратит всего 33 у.е. за 5 месяцев, что при высоком Churn делает юнит-экономику сложной.

## Рекомендации
- Внедрить программу лояльности для повышения Retention
- Улучшить UX на этапе добавления товара в корзину
- Сфокусировать маркетинг на рабочие дни — конверсия выше
- Запустить реактивационные email-кампании для churned пользователей
- Сегментировать пользователей по LTV и работать с крупными покупателями отдельно
